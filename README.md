# 📊 Data Cleaning & Reporting Automation Dashboard

## 🚀 Overview
The Data Cleaning & Reporting Automation Dashboard is an interactive Power BI project that transforms raw sales data into meaningful business insights. Using Power Query for data cleaning and DAX for calculations, the dashboard automates reporting and provides real-time analytics through dynamic visualizations.

## ✨ Key Features
- 🧹 Automated data cleaning and transformation with Power Query
- 📈 KPI Cards for Total Sales, Average Sales, Total Orders, and Products
- 📊 Sales trend analysis by year
- 🌍 Region-wise sales performance comparison
- 🛍️ Category-wise sales distribution
- 🏆 Top-performing products analysis
- 🎛️ Interactive slicers for Region, Category, and Ship Mode
- ⚡ Dynamic and user-friendly dashboard

## 🛠️ Tech Stack
- Power BI Desktop
- Power Query
- DAX
- Microsoft Excel

## 📌 Business Insights
- Monitor overall sales performance.
- Compare sales across different regions.
- Identify the best-performing product categories.
- Discover top-selling products.
- Generate automated reports for faster decision-making.
